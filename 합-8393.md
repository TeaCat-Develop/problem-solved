# 문제 제목:

합

## 문제 정보

- **출처:** https://www.acmicpc.net/problem/8393
- **난이도:** Easy

## 문제 설명

1부터 n까지 합을 출력한다.

## 해결 과정

### 접근 방법

for 반복문을 사용하여 n까지의 범위 내의 정수를 하나씩 더하여 복합연산자에 재할당.
n번째까지 모두 진행된 다음 결과값을 출력하려 함.

점화식을 이용하면 훨씬 간단하게 풀 수 있다는 사실을 알았음.
같은 결과를 얻는 방법에 따라 시스템의 부담을 덜 수 있음을 알게 됨.

## 코드

```python
n = int(input())
for i in range(n):
    n += i
print(n)
```
